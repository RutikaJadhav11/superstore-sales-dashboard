# Superstore Sales Performance Dashboard

## Project Overview

Interactive Power BI dashboard analyzing sales, profitability, products, regions, and customer segments using Superstore data.

## KPIs

- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- YoY Growth %

## Dashboard Features

### Sales Overview
- Monthly Sales Trend
- Top 5 Cities by Sales
- Sales by Category
- Sales by Segment
- Sales by Region

### Profitability Analysis
- Top States by Profit
- Sales vs Profit by Sub-Category
- Top Profit Products
- Top Loss-Making Products

### Detail View
- Product-Level Analysis
- Sub-Category Analysis
- Detailed Sales and Profit Table

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Insights

- California generated the highest profit.
- Technology category achieved the highest sales.
- Consumer segment contributed the most revenue.
- Several high-sales products generated low or negative profit.
- West region outperformed other regions in total sales.
